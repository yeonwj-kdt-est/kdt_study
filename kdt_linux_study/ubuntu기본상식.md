# 기본 상식

## code .은 현재폴더에서 vscode실행

### 명령어 상식

**폴더 생성**

ex) mkdir 사용할 폴더명

- example

    ```bash
    mkdir test
    ```

**폴더 삭제**

ex) rm -rf 삭제할 폴더명

- example

    ```bash
    rm -rf test
    ```

**하위 파일 및 폴더 확인**

```bash
ls
```

**하위 폴더로 접속**

ex) cd 접속할 하위 폴더

- example

    ```bash
    cd test
    ```

**상위 폴더로 접속**

ex) 해당 폴더에서 상위 폴더로 접속

- example

    ```bash
    cd ..
    ```
