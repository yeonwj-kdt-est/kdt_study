# marktext

마크다운을 편리하게 사용할 수 있게 만드는 툴

- [link](https://github.com/marktext/marktext)

## 

## 기본 설명

- `#`은 제목

```bash
# 제목
```

- `##`은 `#`보다 크기 작은 제목

```bash
## subtitle
```

- `###`은 `##`보다 크기 작은 제목

```bash
### middletitle
```

- `######`까지 위 내용 동일

- **``**
  
  - `#`와 같이 하이라이트 주기 위함



- `>`

> 위의 단어로 작성시 해당 항목처럼 표시
> 
> marktext에서는 Quote block사용



- `-`
