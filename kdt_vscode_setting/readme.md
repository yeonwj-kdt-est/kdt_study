# vscode 기본 설치 추천 프로그램

## Extensions에서 확장 프로그램 설치

**Korean Language Pack for Visual Studio Code**

- vscode을 한국어로 사용 가능하게 해주는 확장 프로그램

**Prettier - Code formatter**

- code를 구분가능하고 이쁘게 보이게 해주는 확장 프로그램

**Material Icon Theme**

- 확장 프로그램 아이콘을 ui로 이해하기 쉽게 표시하는 테마 확장 프로그램

**Python**

- python을 vscode에서 실행시키기 위한 확장 프로그램

**Jupyter**

- jupyter를 vscode에서 실행시키기 위한 확장 프로그램

**wsl**

- wsl환경에서 vscode를 실행시키기 위한 확장 프로그램
